##  What it does
A simple GUI tool built with Python + Tkinter. Farmers enter their costs, harvest amount and selling price — it automatically calculates total cost, revenue, and profit/loss.

##  SDG link
- **SDG 2 Zero Hunger**: helps plan better farming
- **SDG 1 No Poverty**: makes income tracking easy

## How to run
1. Install Python 3.x
2. Open `FARM_TRACKER.py` in PyCharm
3. Click Run
4. Fill in your details and click *Calculate*
